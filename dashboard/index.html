Quick Start Guide

                

                    

                        
1

                        

                            
API Access

                            
Your API key is active. Authenticate requests with X-API-Key: <your-key> or use JWT via POST /auth/login.

                        

                    

                    

                        
2

                        

                            
Upload Telemetry CSV

                            
Drop a wide-format CSV above (timestamp + parameter columns). Enter a Satellite ID, click Upload Data, then click Run Analysis to detect anomalies.

                        

                    

                    

                        
3

                        

                            
Review Anomalies

                            
Switch to the Monitor tab — anomalies are listed by severity and confidence. Use the satellite filter to scope by spacecraft.

                        

                    

                    

                        
4

                        

                            
Tune Thresholds

                            
Visit the Channels tab to override z-score, CUSUM, EWMA, and variance thresholds per parameter. Changes take effect on the next analysis run.

                        

                    

                    

                        
5

                        

                            
Configure Alerts

                            
Set a webhook URL or SMTP server in the Alerts tab to receive real-time notifications. For live streaming, push JSON via POST /api/v1/telemetry.
